# Error Log

记录助手的失误模式，用于自我改进。
由 /session-notes 自动写入，/memory-evolution 负责晋升。

## 状态说明
- monitoring：已记录，观察中（出现次数 < 2）
- pending：出现 >= 2 次，等待 evolve.py 晋升到 AGENTS.md
- promoted：已晋升写入 AGENTS.md
- resolved：已修正，不再出现
